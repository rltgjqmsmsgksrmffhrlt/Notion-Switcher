Notion Switcher는 워크스페이스 전환을 2초 이내로 줄여주는 Chrome 확장 프로그램입니다. `Alt+N`을 누르고 숫자키를 누르면 끝.

각 워크스페이스에는 다른 프로젝트, 다른 역할, 다른 꿈이 담겨 있습니다. 전환이 빨라지면 만드는 시간이 늘어나고 — 거기서 진짜 변화가 시작됩니다.

---

## 주요 기능

### 즉시 전환
- 숫자 `1`~`9`로 워크스페이스 즉시 이동
- `Enter`로 첫 번째 검색 결과 열기
- `Shift+클릭`으로 새 탭에서 열기
- 이름, URL, 이모지로 실시간 검색

### 두 가지 뷰
- **팝업** (`Alt+N`) — 320px 컴팩트 패널로 빠른 전환
- **대시보드** (`Alt+Shift+N`) — 전체 페이지 그리드 + 큐브 애니메이션

### 정리
- **폴더** — 워크스페이스를 주제별로 분류
- **드래그 & 드롭** — 순서 변경, 폴더 간 이동
- **추가 / 편집 / 삭제** — 이름, URL, 이모지, 색상 지정

### 개인화
- **테마** — 시스템 / 라이트 / 다크
- **단축키 커스텀** — 충돌 감지 기능 포함
- **다국어** — 브라우저 언어에 따라 한국어/영어 자동 전환

---

## 단축키

| 단축키 | 동작 | 범위 |
|--------|------|------|
| `Alt+N` | 팝업 열기 | 전역 |
| `Alt+Shift+N` | 대시보드 열기 | 전역 |
| `1`~`9` | 해당 번호 워크스페이스로 이동 | 팝업 / 대시보드 |
| `Enter` | 첫 번째 결과 열기 | 검색 시 |
| `Escape` | 검색 초기화 / 닫기 | 팝업 / 대시보드 |
| `/` | 검색창 포커스 | 대시보드 |
| `D` | 대시보드 열기 | 팝업 (검색창 비어있을 때) |

## 설치

### Chrome Web Store
준비 중

### 소스에서 설치 (개발자 모드)
1. `chrome://extensions` 접속
2. **개발자 모드** 활성화
3. **압축해제된 확장 프로그램을 로드합니다** 클릭
4. `notion-switcher` 폴더 선택
